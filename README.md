# 招投标采购平台原型

这是一个招投标采购平台的原型项目，用于演示招标、投标、评标、定标、合同归档等核心业务流程。

## 技术栈

- React 18
- React Router 6
- Ant Design 5
- Vite

## 角色

平台支持以下角色：

- 招标人
- 招标代理
- 投标人/供应商
- 评标专家
- 监督人员
- 平台管理员

## 快速开始

```bash
npm install
npm run dev
npm run build
```

## 项目结构

- `src/views/` — 页面组件
- `src/components/` — 通用组件
- `src/hooks/` — React Hooks
- `src/router/` — 路由配置
- `src/config/` — 权限与常量配置
