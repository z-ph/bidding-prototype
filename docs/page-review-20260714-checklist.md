# 页面评审验证清单

本文件汇总 `page-reviews-20260714-0031.zip` 和 `2026-07-14-yy0-前端页面评审结果.md` 两份评审意见的验证方法，供人工复核或 agent 回归测试使用。

验证原则：
- 不能只看“有没有这个页面/按钮”，要看**在正确角色、正确状态下是否表现出正确行为**。
- 每个条目必须给出：测试角色、前置状态、操作步骤、预期结果、是否通过。
