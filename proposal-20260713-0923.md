# 变更提案：优化门户首页交易信息筛选区域

- 提案编号：proposal-20260713-0923

- 状态：已实施

## 问题背景

门户首页 `/bidding-prototype/`（`src/views/Portal.vue`）的“交易信息”区块存在两处体验问题：

1. **菜单栏位置问题（高）**：筛选菜单（全部 / 招标公告 / 变更公告 / 候选人公示 / 中标公告）目前放在内容区标题行，不符合顶部导航的常规布局，视觉上与页面标题混杂。
2. **图标/标签太小（中）**：同一区域的筛选按钮尺寸偏小，不够醒目。

## 目标

将交易信息筛选菜单上移至页面顶部导航区域，并适当增大标签尺寸，使门户首页的信息筛选更符合顶部导航/页签的常规交互模式。

## 方案

1. 在 `portal-header` 下方新增顶部筛选导航栏（`.filter-nav`）。
2. 将 `noticeType` 的 `el-radio-group` 从 `div.section-title` 中移除，迁移到新的顶部筛选导航栏。
3. 顶部筛选导航使用较大的按钮尺寸（`size="large"` 或等效样式），提升可点击性与视觉层级。
4. `div.section-title` 仅保留“交易信息”标题，右侧可保留分页等辅助操作。
5. 保持现有响应式数据与筛选逻辑不变。

## 涉及文件

- `src/views/Portal.vue`

## 验证

- 运行 `npm run build` 构建通过，无错误。
- 门户首页顶部出现交易信息筛选页签。
- 点击不同页签可正确切换公告类型列表。
