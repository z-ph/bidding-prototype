# Proposal: 补齐“招聘会”负责但当前原型未覆盖的功能

## Change ID

`add-portal-org-missing-features`

## Why

根据 `2026-07-14-招投标采购平台-逐功能开发任务清单.xlsx`，实际开发执行人为“招聘会”的任务共 18 项。逐条比对当前 React UI 原型（`src/views/`、`src/router/index.jsx`、`src/components/Layout.jsx`、`src/config/permissions.js`）后发现：部分任务仅有简化展示或占位按钮，缺少完整的管理后台/交互功能。本提案将这些未覆盖点补齐，使原型与任务清单对齐。

## What Changes

1. **门户内容管理后台**（PORTAL-003）
   - 新增管理员“新闻公告维护”页面：分类、正文、附件、草稿、发布/下线。
   - 在现有 `/news` 公共列表页接入真实数据与详情页。

2. **帮助中心搜索**（PORTAL-004）
   - 在现有 `/help` 页面增加关键词搜索与分类筛选。

3. **部门树维护**（ORG-006）
   - 在 `/admin/organization` 实现部门新增、编辑、删除、拖拽排序，不再只是占位弹窗。

4. **子账号管理**（ORG-007）
   - 新增 `/admin/sub-accounts` 页面：创建、停用、修改联系信息、重置密码。
   - 在 `Layout` 菜单中为管理员角色添加入口。

5. **组织与角色数据范围衔接**（ORG-008）
   - 在组织树节点上配置数据范围（本人/本部门/本企业/全部）。
   - 与现有权限配置联动，控制项目、公告等数据的可见范围。

## Impact

- **新增页面**：新闻公告维护、子账号管理、数据范围配置。
- **增强页面**：现有 `/news`、`/help`、`/admin/organization`。
- **删除/修改**：无现有功能删除，仅扩展。
- **权限配置**：`permissions.js` 增加新页面路由与角色映射。

## Out of Scope

- 真实短信通道与真实 CA 证书驱动（AUTH-004/005 已用演示方式实现，不在本次补齐范围）。
- 首页轮播图（PORTAL-001 已用统计区+快捷入口替代，本次不做）。
- 帮助中心后台文章编辑（若只补齐搜索，后台编辑可另案处理）。
